# Shortee Brand Lexicon

**Date:** 2026-03-08  
**Owner:** Brand / Information Architecture  
**Purpose:** Keep Shortee language precise, repeatable, and hard to dilute.

---

## 1. Allowed Core Nouns

- `피드`
- `장면`
- `자막`
- `표현`
- `시리즈`
- `저장`
- `반복`
- `이어보기`
- `기록`
- `보관함`

---

## 2. Allowed Support Verbs

- `보다`
- `넘기다`
- `멈추다`
- `남기다`
- `저장하다`
- `반복하다`
- `이어보다`
- `다시 보다`
- `듣다`
- `고르다`
- `따라가다`

---

## 3. Use Sparingly

- `학습`
- `연습`
- `복습`
- `향상`
- `발견`
- `습관`
- `진도`
- `루틴`
- `오늘의 표현`

이 단어들은 필요할 때만 보조적으로 쓴다.  
브랜드 최전면에는 두지 않는다.

---

## 4. Banned / Avoid

- `커리큘럼`
- `코스`
- `모듈`
- `루틴`을 hero word로 쓰는 것
- `미션`
- `스트릭`을 브랜드 문장처럼 쓰는 것
- `대시보드`
- `grammar-first`
- `AI tutor`
- `teacher`
- `classroom`
- `test prep`
- `TOEIC`
- `IELTS`
- `영어 정복`
- `점수 향상`

---

## 5. Preferred Rewrites

- `영어 공부 시작` -> `장면부터 보기`
- `학습 루틴 만들기` -> `저장한 표현 다시 보기`
- `오늘의 미션` -> `오늘 볼 장면`
- `학습 대시보드` -> `저장한 표현`
- `영어 실력 향상` -> `영어가 덜 낯설어짐`
- `개인화 커리큘럼` -> `다시 열기 쉬운 피드`
- `학습 시작하기` -> `한 장면 먼저 보기`

---

## 6. Core Rule

Shortee 언어는:

- object-first
- motion-first
- media-first

여야 한다.

즉, 설명보다 장면과 행동이 먼저 보여야 한다.
