## 1. Brand North Star

### Brand role

Shortee는 영어를 "배우게" 만드는 앱이 아니라, **짧은 장면을 자꾸 보게 하고 자막에서 멈추게 해서 영어가 덜 낯설어지게 만드는 브랜드**여야 한다.

### Brand promise

> 보다 보면 귀가 먼저 익숙해진다.

### One-line positioning

**Shortee는 자막, 저장, 반복, 이어보기를 중심으로 영어에 익숙해지는 쇼츠 피드다.**

### Category definition

Shortee는 아래 카테고리에 들어가면 안 된다.

- 시험 대비 앱
- 문법 중심 학습 앱
- AI 튜터 앱
- 생산성 앱

Shortee는 아래 카테고리를 스스로 만들어야 한다.

- 영어에 익숙해지는 쇼츠 피드
- 표현 중심 immersion feed

## 2. Brand Strategy

### Who we serve

핵심 사용자:

- 영어를 해야 한다는 압박은 싫지만 영어 감각은 갖고 싶은 사람
- 드라마, 영화, 음악, 밈, 인터뷰 같은 장면 소비가 익숙한 사람
- 하루 3분에서 15분 사이에 자주 들어오는 사람
- "앉아서 공부"보다 "보다가 남는 것"을 원하는 사람

### User job

사용자는 영어 실력 향상을 구매하지 않는다.  
사용자는 아래 감각을 원한다.

- 영어를 덜 무섭게 느끼는 감각
- 실제 표현이 익숙해지는 감각
- 짧게 자주 들어가도 성취감이 남는 감각

### Competitive enemy

Shortee가 싸워야 하는 상대는 앱 하나가 아니다.

진짜 경쟁 상대:

- "영어 앱은 피곤하다"는 인식
- 진입 첫 화면에서 오는 공부 냄새
- 길고 무거운 세션
- 틀리면 압박하는 보상 구조

### What makes the brand strong

브랜드 강함은 이 네 가지로 정의한다.

1. 한 문장으로 설명된다.
2. 첫 화면 3초 안에 인상이 온다.
3. 다른 영어 앱과 말투가 겹치지 않는다.
4. 광고를 보지 않아도 어떤 브랜드인지 바로 안다.

## 3. Brand Personality

### Personality keywords

- 선명한
- 가벼운
- 재치 있는
- 성숙한
- 빠른

### Not our personality

- 유치한
- 선생님 같은
- 과장된
- 과하게 친한
- 힙해 보이려고 애쓰는

### Tone formula

**짧고 + 건조하지 않고 + 과장하지 않고 + 리듬감 있게**

예시:

- 좋은 톤: "오늘 볼 장면을 골라보세요"
- 나쁜 톤: "오늘도 영어 정복을 위한 여정을 시작해보세요"

## 4. Voice System

### Core copy rules

1. 한 문장에서 한 개의 메시지만 준다.
2. "학습", "커리큘럼", "완성", "정복" 같은 단어를 기본적으로 피한다.
3. 보상보다 감각을 말한다.
4. 명령보다 제안처럼 들리게 쓴다.
5. 설명보다 장면을 앞세운다.
6. `장면` 하나로 모든 문장을 해결하려 하지 않는다.

### Vocabulary to use

- 피드
- 장면
- 자막
- 표현
- 말맛
- 반복
- 귀에 남다
- 익숙해지다
- 바로 보기
- 이어 보기
- 지금 볼 것
- 저장한 표현
- 가볍게
- 오늘 볼 장면

### Vocabulary to reduce

- 장면 앱
- 루틴
- 학습
- 수강
- 커리큘럼
- 문제풀이
- 오답
- 정답률
- 실력 향상

### Do / Don't

| Do | Don't |
|---|---|
| 오늘 볼 장면 | 오늘의 학습 콘텐츠 |
| 표현이 남는 앱 | 영어 실력 향상 솔루션 |
| 가볍게 시작하기 | 체계적으로 공부하기 |
| 이어서 보기 | 이전 학습 이어하기 |
| 저장한 표현 | 암기 문장 |

## 5. Visual Direction

### Current visual diagnosis

현재 화면 인상은 아래와 같다.

- 로고는 강한데 전체 색 체계는 아직 generic하다
- 검정 바탕 + 보라 포인트가 "공부 앱"은 아니게 만드나, 아직 `Shortee만의 색`은 아니다
- 로그인, 탐색, 설정 화면이 각각 나쁘지는 않지만 한 기업의 정교한 브랜드 시스템으로 느껴질 정도로 응집되진 않았다

### Visual problem list

1. 보라 계열이 너무 기본값처럼 보인다.
2. 정보 밀도와 카피 밀도는 좋지만, 브랜드 무드는 아직 "앱 UI"에 머물러 있다.
3. 버튼과 카드의 느낌은 안정적이지만, 브랜드를 기억시키는 한 방이 약하다.

### Recommended direction

Shortee는 아래 방향으로 가야 한다.

- 배경: 거의 검정에 가까운 짙은 톤 유지
- 주조색: 보라 대신 더 독자적인 축으로 이동
- 포인트: 빠른 리듬, 선명한 대비, 장면 중심 인상
- 형태: 둥글지만 귀엽지 않게
- 타이포: 강한 워드마크 + 단정한 본문

### Recommended palette direction

권장 주조색 축:

- deep teal / cold jade
- graphite black
- warm off-white
- signal amber for reward only

이 방향이 맞는 이유:

- 현재 보라보다 경쟁 앱과 덜 겹친다
- 밤 시간 숏폼 소비 감각과 잘 맞는다
- 성숙하고 선명한 무드를 만든다

### Visual non-negotiables

- 귀여운 캐릭터 중심 브랜드로 가지 않는다
- 보라 그라데이션 기반 generic startup look으로 가지 않는다
- 라이트 모드를 메인 정체성으로 두지 않는다
- "교육 SaaS"처럼 보이는 카드 UI로 가지 않는다

## 6. Brand Architecture

### Brand hierarchy

- Master Brand: `Shortee`
- Product descriptor: `영어에 익숙해지는 쇼츠 피드`
- Product descriptor: `자막, 저장, 반복으로 이어보는 영어 피드`
- Core product nouns:
  - 피드
  - 장면
  - 자막
  - 표현
  - 시리즈
  - 저장
  - 반복
  - 이어보기

### Naming rules

같은 개념은 같은 단어만 쓴다.

권장:

- Explore -> 탐색 또는 둘러보기 중 하나로 통일
- Learning -> 복습 또는 내 기록 중 하나로 통일
- Profile / Settings -> 설정으로 통일, 프로필은 계정 단위에서만 사용

### Proposed product labels

하단 탭 기준 권장안:

- `오늘`
- `피드`
- `보관함`
- `설정`

이유:

- 현재 `쇼츠` 탭은 기능명이지 브랜드 경험명이 아니다
- `오늘`은 재방문 루프를 만든다
- `보관함`은 저장 표현과 기록을 함께 담기 더 정확하다

### Brand ownership note

브랜딩 팀은 아래를 소유한다.

- 브랜드 문장
- 용어 체계
- 기본 색 축
- 금지어
- 캠페인 메시지

UI/UX 팀은 아래를 소유한다.

- 레이아웃
- 인터랙션
- 컴포넌트 상세 구조
- 정보 배치
- 모션 디테일

즉, 브랜딩 팀은 "무슨 인상과 단어를 써야 하는가"를 고정하고, UI/UX 팀은 "그 인상을 어떤 화면 구조로 구현할 것인가"를 푼다.

## 7. Messaging Architecture

### Brand message ladder

#### Level 1: identity

자막, 저장, 반복으로 이어보는 영어 피드

#### Level 2: value

- 짧아서 들어오기 쉽다
- 자꾸 보다 보면 덜 낯설어진다
- 자막에서 멈추고 저장하고 반복해서 다시 보기 좋다

#### Level 3: proof

- 짧은 클립 중심
- 자막 상호작용
- 표현 저장
- 시리즈 이어보기
- 가벼운 반복과 다시 보기

### Landing / ad message buckets

#### Bucket A: entertainment-first

- 짧게 넘기는데 말맛은 남습니다
- 보다 보면 귀가 먼저 익숙해집니다

#### Bucket B: low-friction

- 길게 앉지 않아도 피드는 끊기지 않습니다
- 자막에서 멈추고 다시 보다 보면 영어가 덜 낯설어집니다

#### Bucket C: expression payoff

- 저장한 표현이 나중에 더 빨리 들립니다
- 설명보다 먼저 들리는 영어가 있습니다

## 9. Campaign System

### Launch campaign thesis

Shortee의 첫 캠페인은 "영어를 열심히 하는 사람"을 부르면 안 된다.  
"장면을 재밌게 보는 사람"을 불러야 한다.

### Campaign theme candidates

1. **영어는 장면으로 남는다**
2. **보다 보면 귀가 먼저 익숙해진다**
3. **짧게 넘기는데 말맛은 남는다**

### Recommended launch theme

**영어는 장면으로 남는다**

이유:

- 포스터/스토어/광고에서 한 줄 인상이 가장 강하다
- 광고, 앱스토어, 랜딩, SNS까지 하나로 묶기 쉽다
- 성숙하고 과장되지 않는다

단, 제품 기본 설명문까지 이 문장 하나로만 밀지는 않는다.  
제품 설명은 `보다 보면 귀가 먼저 익숙해진다` 축으로 더 구체적으로 쓴다.

### Campaign formats

- 6초 후킹 영상
- 장면 + 표현 카드
- Before/after 카피 실험
- 한 장면 한 표현 시리즈

## 10. Current Brand Gaps

### Highest-priority brand gaps

1. `Shortee`와 `StudyEng`가 공존한다
2. 기본 색 체계가 아직 `Shortee 고유 인상`을 만들지 못한다
3. 화면 라벨이 한국어/영어 혼합으로 일관성이 부족하다
4. 공유물이 브랜드 자산으로 확장되지 않는다
5. 광고 메시지용 카피 뱅크가 아직 없다

### What branding team should own next

1. 최종 색 체계 확정
2. 최종 라벨 체계 확정
3. 앱스토어용 브랜드 카피 세트 제작
4. SNS/광고용 표현 카드 시스템 제작
5. 브랜드 표현 가이드와 금지어 사전 운영

## 12. Non-Goals

이 문서는 아래를 다루지 않는다.

- 보안
- 결제 안정성
- 배포 안정성
- 성능 최적화
- 인프라

그 영역은 다른 팀장이 소유한다.  
브랜딩 팀은 **브랜드 인상, 언어, 구조, 캠페인, 자산**에만 집중한다.
