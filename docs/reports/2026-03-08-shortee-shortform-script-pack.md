# Shortee Short-Form Script Pack

**Date:** 2026-03-08  
**Owner:** Brand / Growth Lead  
**Use:** Instagram Reels, YouTube Shorts, creator UGC, paid social

---

## 1. Rule

Shortee 영상은 기능 설명 영상처럼 시작하면 안 된다.

항상 아래 순서를 기본으로 한다.

1. 장면 또는 한 줄
2. 왜 귀에 남는지
3. Shortee 안에서 어떻게 이어지는지

즉:

`scene -> subtitle/line -> repeat/save -> CTA`

---

## 2. 15-Second Script Set

### 1. Scene Replay

- Hook: `왜 이 장면은 한 번 더 보게 될까요?`
- Middle: 같은 대사를 한 번 더 듣고, Shortee에서 반복 재생하는 흐름을 보여준다.
- CTA: `이 장면, Shortee에서 다시 보기`
- Fit: 반복과 장면 몰입을 가장 직접적으로 보여준다.

### 2. Expression You Actually Hear

- Hook: `교과서보다 이런 표현을 더 자주 듣습니다`
- Middle: 실제 장면 속 한 줄을 보여주고 저장 흐름으로 연결한다.
- CTA: `저장해두고 다시 듣기`
- Fit: 표현 저장과 실제성 강조에 맞다.

### 3. From Scroll to Repeat

- Hook: `넘기다 보니 다시 듣게 되는 장면`
- Middle: 피드에서 넘기고, 자막에서 멈추고, 반복하는 흐름을 빠르게 보여준다.
- CTA: `오늘 피드 열기`
- Fit: Shortee의 실제 사용 리듬을 잘 보여준다.

### 4. No Lecture, Just Lines

- Hook: `설명보다 먼저 귀에 남는 영어`
- Middle: 장면, 자막, 번역, 다시 듣기 순서로 감각을 전달한다.
- CTA: `한 줄부터 다시 보기`
- Fit: 공부앱 말투 없이도 제품 가치를 설명할 수 있다.

### 5. Drama Taste Angle

- Hook: `드라마 좋아하면 넘길 장면이 끊기지 않습니다`
- Middle: 감정선 있는 장면 2-3컷을 연결하고 이어보기 흐름으로 간다.
- CTA: `취향대로 이어보기`
- Fit: 취향 기반 유입에 강하다.

### 6. The Line After the Line

- Hook: `이 표현이 남는 이유는 나중에 한 번 더 들리기 때문입니다`
- Middle: 처음 나온 장면과 다시 나온 장면을 붙여 보여준다.
- CTA: `반복해서 더 빨리 듣기`
- Fit: 반복과 귀 익숙해짐의 논리를 설명한다.

### 7. Office Small Talk

- Hook: `회사에서 진짜 많이 듣는 말`
- Middle: 상황이 바로 이해되는 짧은 장면과 자막을 보여준다.
- CTA: `실제 장면으로 듣기`
- Fit: 상황형 소재에 적합하다.

### 8. One Scene, One Phrase

- Hook: `한 장면이면 한 표현은 남습니다`
- Middle: 하나의 장면, 하나의 대사, 하나의 저장 흐름만 짧게 보여준다.
- CTA: `오늘 한 장면 보기`
- Fit: 15초 광고에 가장 압축적인 구조다.

### 9. Subtitle Brain vs Real Ear

- Hook: `읽는 영어와 들리는 영어는 다릅니다`
- Middle: 텍스트 번역보다 장면 속 말맛이 더 먼저 오는 순간을 보여준다.
- CTA: `자막과 같이 다시 듣기`
- Fit: 자막 중심 제품 구조를 직접 드러낸다.

### 10. Addictive Little Rewatch

- Hook: `억지로 외우지 않아도, 자꾸 다시 보게 됩니다`
- Middle: 같은 짧은 장면을 만족감 있게 반복 재생하는 모습으로 간다.
- CTA: `다시 보고 남기기`
- Fit: low-friction return loop를 가장 자연스럽게 전달한다.

### 11. Familiarity Buffer

- Hook: `자주 들으면 낯선 말도 덜 낯설어집니다`
- Middle: 비슷한 표현이 다른 장면에서 반복 노출되는 감각을 보여준다.
- CTA: `피드에서 익숙해지기`
- Fit: 과장 없이 귀 익숙해짐을 설명한다.

### 12. What To Watch Next

- Hook: `한 장면 끝나면 다음 장면까지 바로 갑니다`
- Middle: 시리즈/추천 흐름으로 다음 클립이 이어지는 모습을 보여준다.
- CTA: `다음 장면 이어보기`
- Fit: Shortee의 큐 기반 구조를 드러낸다.

## 3. Edit Rule

### Structure

- 0-2초: 강한 장면 또는 문장
- 2-8초: 자막/표현/반복/저장 중 하나 강조
- 8-12초: 앱 안 흐름 연결
- 12-15초: CTA

### Keep

- 자막 크게
- 장면과 표정 살리기
- 컷 간 리듬 빠르게

### Avoid

- 앱 기능 나열
- 과한 전환 효과
- 선생님 톤 내레이션
- 긴 서론

---

## 4. Best Angles To Test First

1. `반복해서 더 빨리 들리는가`
2. `저장한 표현이 남는가`
3. `취향 장면을 계속 넘기게 되는가`
4. `자막과 함께 귀가 익숙해지는가`
