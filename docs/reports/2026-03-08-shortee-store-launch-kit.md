## 1. Decision

스토어 기준 브랜드는 아래로 고정한다.

- 브랜드명: `Shortee`
- 브랜드 약속 문장: `보다 보면 귀가 먼저 익숙해집니다`
- 캠페인 문장: `영어는 장면으로 남는다`
- 카테고리 인식: `영어 공부 앱`보다 `자막, 저장, 반복으로 이어보는 영어 피드`
- 톤: 짧고 선명하게, 선생님 말투 없이

스토어에서 가장 먼저 보여야 하는 것은 기능 목록이 아니라 브랜드 인상이다.

## 2. Platform Rules Checked

### App Store

- 앱 이름: 최대 30자
- 부제: 최대 30자
- 프로모션 텍스트: 최대 170자
- 설명: 최대 4000자
- 키워드: 최대 100 bytes
- 스크린샷: 최대 10장
- 앱 프리뷰: 각 로컬라이제이션/기기 크기별 최대 3개

### Google Play

- 앱 이름: 최대 30자
- 짧은 설명: 최대 80자
- 전체 설명: 최대 4000자
- 스토어 게시를 위해 최소 2장 이상의 스크린샷 필요
- 프리뷰 비디오는 YouTube URL 1개 연결 가능

## 3. Store Positioning Rule

스토어에서 Shortee는 이렇게 보여야 한다.

- `영어 실력 향상 솔루션`처럼 말하지 않는다
- `드라마/영화/밈/인터뷰 장면을 넘기다 보면 귀가 익숙해지는 앱`처럼 보이게 만든다
- 첫 3개 스크린샷 안에 브랜드 문장, 장면성, 저장/복습 구조를 모두 담는다
- 텍스트는 짧고, 장면 중심 명사로 끊는다

금지:

- 과한 교육어
- 성과 과장
- 시험 대비 인상
- generic한 AI 영어앱 문장

## 4. Final Metadata Pack

### 4.1 App Store KR

#### Name

`Shortee`

- 문자 수: 7 / 30

#### Subtitle

`보다 보면 귀가 익숙해진다`

- 문자 수: 12 / 30

#### Promotional Text

`짧은 장면을 넘기고 자막에서 멈추다 보면 귀가 먼저 익숙해지는 영어 피드. 저장한 표현은 나중에 더 빨리 다시 들립니다.`

- 문자 수: 64 / 170

#### Keywords

`영어숏폼,영어표현,영어리스닝,미드영어,영어쉐도잉,영어회화`

- UTF-8 bytes: 83 / 100
- 주의: App Store Connect는 키워드를 bytes 기준으로 계산하므로, 로컬라이제이션 추가 시 다시 확인한다.

#### Description

```text
영어를 공부하려고 들어오는 앱이 아니라, 짧은 장면을 넘기고 자막에서 멈추다 보면 귀가 먼저 익숙해지는 앱.

Shortee는 짧은 장면을 넘기다가 표현을 남기고, 다시 이어보게 되는 영어 피드입니다.

이런 사람에게 맞습니다.
- 길고 무거운 영어 공부 앱은 부담스러운 사람
- 드라마, 영화, 인터뷰, 밈처럼 장면으로 영어를 익히고 싶은 사람
- 하루 3분에서 15분 사이에 자주 들어오는 사람
- 외우기보다 자주 보고 자연스럽게 남기고 싶은 사람

Shortee에서 할 수 있는 것
- 오늘 볼 장면 바로 보기
- 짧은 클립에서 자막과 표현 붙잡기
- 마음에 남는 표현 저장하기
- 저장한 표현 다시 꺼내보기
- 시리즈로 이어보며 감각 유지하기

Shortee는 이렇게 다릅니다.
- 들어오면 공부보다 장면이 먼저 보입니다.
- 진입이 가볍고 다시 들어오기가 쉽습니다.
- 저장과 복습이 무겁지 않습니다.
- 선생님 말투보다 실제 장면의 리듬을 살립니다.

보다 보면 귀가 먼저 익숙해집니다.
지금 Shortee에서 오늘 장면을 바로 보세요.
```

## 6. Preview Video Rule

### App Store

- 첫 출시에서 앱 프리뷰를 억지로 넣지 않는다.
- 이유: App Store는 앱 프리뷰가 없을 때 첫 1-3개 스크린샷이 검색 결과에서 더 직접적으로 보일 수 있다.
- 브랜드가 약한 소개 영상보다 강한 스크린샷 3장이 더 중요하다.

### Google Play

- 프리뷰 비디오는 YouTube URL 1개만 연결한다.
- 정제된 20-30초 브랜드 컷이 준비되기 전까지는 비워두는 것이 낫다.
- 이유: Play는 프리뷰 비디오가 있을 경우 스크린샷보다 먼저 보이는 영역이 생길 수 있다.

### Preview storyboard when ready

1. 0-3초: `보다 보면 귀가 먼저 익숙해집니다`
2. 3-7초: 오늘 홈에서 장면 선택
3. 7-12초: 쇼츠 장면 몰입
4. 12-16초: 표현 저장
5. 16-21초: 복습 보드
6. 21-25초: `지금 Shortee에서 오늘 장면 받기`

## 7. Store Asset Notes

### Visual tone

- dark-first
- jade / deep teal accent
- 보라 중심 generic 인상 금지

### Screenshot writing rule

- 한 장에 메시지 하나
- 길게 설명하지 않는다
- 기능보다 감각을 앞세운다
- 모든 문장은 광고 소재로 재사용 가능한 수준으로 짧아야 한다

### Icon direction

- 워드마크나 앱 심볼이 있더라도 `교육앱 느낌`보다 `짧고 또렷한 미디어 앱 느낌`이 나야 한다
- 캐릭터 중심 아이콘 금지

## 8. Brand-Owned Release Checklist

브랜딩 팀이 출시 전 확인할 것:

1. App Store / Play 브랜드명이 모두 `Shortee`인지
2. 첫 문장이 `보다 보면 귀가 먼저 익숙해집니다` 축을 유지하는지
3. 첫 3개 스크린샷이 장면성, 저장, 복습 구조를 보여주는지
4. subtitle / short description이 공부앱처럼 들리지 않는지
5. 설명문과 광고 문장이 같은 브랜드처럼 들리는지
6. 스토어 비주얼이 dark + jade 축으로 통일되는지
