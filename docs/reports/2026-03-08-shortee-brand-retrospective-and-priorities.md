# Shortee Brand Retrospective And Priorities

**Date:** 2026-03-08  
**Owner:** Brand Lead  
**Purpose:** Look back at the branding work so far, keep what is sharp, cut what is bloated, and lock the next priority edits.

---

## 1. What We Got Right

### 1. We refused the default edtech frame

가장 중요한 판단은 처음부터 맞았다.

- `시험/문법/강의` 축으로 가지 않기
- `공부 냄새`를 줄이기
- `media-like` 감각을 키우기

이 방향이 맞았기 때문에 Shortee는 최소한 평범한 영어 앱처럼 보이지 않게 되었다.

### 2. We found the real visual center

쇼츠 플레이어, 자막, 저장, 반복이 실제 브랜드 중심 자산이라는 점을 잡은 것도 맞았다.

지금 브랜드다운 순간은:

- player
- subtitle
- repeat
- save

이고,

브랜드답지 않은 순간은:

- dashboard
- stats
- mission
- generic card stack

이다.

### 3. We separated campaign from product copy

`영어는 장면으로 남는다`는 좋은 문장이다.  
하지만 모든 곳의 첫 문장으로 쓰기엔 추상적이라는 점을 뒤늦게라도 정리한 건 맞았다.

현재 분리는 타당하다.

- 캠페인 문장: `영어는 장면으로 남는다`
- 제품 약속 문장: `보다 보면 귀가 먼저 익숙해집니다`
- 제품 설명 문장: `자막, 저장, 반복, 이어보기로 영어가 덜 낯설어지는 피드`

### 4. We moved from noun-only branding to behavior branding

초반에는 `장면`이라는 명사 하나에 많이 기대고 있었다.

지금은 아래 행동 단어가 더 중요하다는 걸 잡았다.

- 넘기다
- 멈추다
- 남기다
- 반복하다
- 이어보다

이건 좋은 진전이다.

---

## 2. What We Overdid

### 1. We made too many manifesto-like lines

초기 문서와 카피는 한동안 `포스터 문장`이 많았다.

문제:

- 브랜드는 선명해 보여도
- 실제 제품 설명은 약해진다
- 오그라드는 위험이 높아진다

원칙:

슬로건은 적게, 행동 설명은 더 많이.

### 2. We overused contrast language

`공부 앱 말고`, `영어 앱은 피곤하다` 같은 문장은 한두 번은 먹힌다.  
하지만 많이 쓰면 Shortee가 자기 언어를 가진 브랜드가 아니라, 경쟁사 반사체처럼 보인다.

원칙:

반대 진술보다 사용자 체감 진술을 우선한다.

### 3. We let too many structural nouns leak into the front

`루틴`, `복습`, `대시보드`, `오늘 상태` 같은 단어는 구조상 쓸 수 있지만,  
브랜드 전면으로 오면 제품이 다시 productivity app처럼 보인다.

원칙:

- front = 피드 / 자막 / 표현 / 반복 / 이어보기
- back = 기록 / 상태 / 통계 / 운영

### 4. We still have some vocabulary drift

지금도 남아 있는 충돌:

- `복습` vs `보관함`
- `나` vs `설정`
- `오늘`의 과사용
- `루틴`의 과의미화
- 스토어 문서 안 `복습 보드` 같은 잔여어

이건 아직 정리 중이다.

---

## 3. What We Learned About Shortee

Shortee는 `짧은 장면 앱`이 아니다.

더 정확히는:

**짧은 장면을 자막과 함께 넘기다가, 필요한 표현을 남기고, 다시 돌아오게 되는 영어 피드**

이 문장에서 중요한 건 순서다.

1. 본다
2. 멈춘다
3. 남긴다
4. 다시 온다

해자는 이 순서에서 나온다.

---

## 4. What To Keep Firm

### Keep without debate

- `Shortee`
- dark-first + jade
- player-first visual hierarchy
- subtitle interaction as a core product asset
- campaign line `영어는 장면으로 남는다`
- product promise `보다 보면 귀가 먼저 익숙해집니다`

---

## 5. What To Revisit Next

### Priority 1: navigation noun system

다음 결정을 빨리 고정해야 한다.

- `오늘`
- `피드`
- `보관함`
- `설정`

이 축으로 갈지, 일부를 더 바꿀지.

현재로서는 이 구성이 가장 낫다.

### Priority 2: review language cleanup

`복습`은 기능 설명으로는 쓸 수 있지만,  
탭/스토어/브랜드 전면어로는 신중해야 한다.

다음 후보:

- 보관함
- 다시 보기
- 저장 표현

### Priority 3: store screenshot text tightening

스토어 문서에 아직 남아 있는 설명형 표현을 더 줄여야 한다.

특히:

- `오늘 장면 받기`
- `복습 보드`
- `구조`

같은 표현은 더 자연스럽게 바꿔야 한다.

### Priority 4: README and root docs consistency

루트 문서도 이제 product truth에 더 가까워졌지만,  
다음엔 `learning app` 같은 잔여 표현을 완전히 없애야 한다.
