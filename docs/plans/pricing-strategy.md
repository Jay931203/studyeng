**한국 시장 특성:**
- 영어 학습 앱 시장 매년 약 50% 성장 중
- 말해보카, 스픽이 교육 카테고리 매출 1, 2위

**유저 심리:**
- 월 10,000원 미만 = "부담 없다"
- 월 15,000~20,000원 = "투자하는 느낌"
- 월 25,000원+ = "비싸다, 효과 확실해야 함"
- 연간 결제 시 "2개월 무료" 수준의 할인이 전환율 최적화 (업계 표준)

## 2. StudyEng 추천 가격 체계

### 2.1 핵심 포지셔닝

> **"케이크보다 재밌고, 듀오링고보다 실용적이고, 스픽보다 부담 없다"**

StudyEng은 동영상 기반 + 게임형 복습이라는 독자적 포지션.
AI를 게임 엔진으로 활용하는 독특한 구조이므로, **케이크와 듀오링고 사이**의 가격대가 적절.

### 2.2 구독 플랜 구성

#### Free (무료)
- 가격: 0원
- 역할: 유저 유입, 제품 체험, 바이럴 확산

#### Premium Monthly (월간)
- **가격: 9,900원/월**
- 역할: 진입 장벽 낮추기, 연간 플랜의 가격 앵커

#### Premium Quarterly (3개월)
- **가격: 23,900원/3개월 (월 환산 7,967원, 20% 할인)**
- 역할: 월간 → 연간 사이 징검다리, "한번 해볼까" 유저 전환

#### Premium Annual (연간) -- 메인 추천
- **가격: 69,900원/년 (월 환산 5,825원, 41% 할인)**
- 역할: 핵심 수익원, "Best Value" 태그 표시
- 메시지: "월 5,825원 -- 커피 한 잔보다 저렴한 영어 학습"

#### Premium Lifetime (평생)
- **가격: 199,000원 (1회 결제)**
- 역할: 가격 앵커 + 충성 유저 보상 + 초기 현금흐름 확보
- 한정 판매 (런칭 기간만 or 이벤트 시에만 오픈)

### 2.3 가격 설계 근거

```
월간 9,900원 x 12 = 118,800원
연간 69,900원 → 48,900원 절약 (41% 할인)
→ "2개월 무료" 보다 큰 체감 할인 → 연간 전환 유도

3개월 23,900원 x 4 = 95,600원
연간 69,900원 → 25,700원 절약 (27% 추가 할인)
→ 3개월 유저도 연간으로 업그레이드 유도

평생 199,000원 ÷ 69,900원 = 2.85년분
→ 3년 이상 쓸 유저에게 합리적
→ 연간 대비 "비싸 보이게" 만들어 연간이 합리적으로 느껴지게
```

## 3. Free vs Premium 기능 비교

| 기능 | Free | Premium |
|---|---|---|
| **일일 동영상 시청** | 5개/일 | 무제한 |
| **자막 모드** | 영어만 | 영어 + 한국어 동시 |
| **구문 저장** | 10개 | 무제한 |
| **복습 게임** | 기본 1종 | 전체 5종+ |
| **광고** | 있음 | 없음 |
| **AI 게임 엔진** | 기본 난이도 | 개인화 난이도 조절 |
| **학습 통계** | 기본 (오늘만) | 상세 (주간/월간 리포트) |
| **오프라인 저장** | 불가 | 가능 |
| **콘텐츠 카테고리** | 기본 카테고리만 | 전체 카테고리 |
| **스트릭 보호** | 불가 | 월 2회 보호 |

### 무료 티어 설계 철학

> "무료에서도 재미를 느끼되, 더 많이 하고 싶게 만든다"

- 5개/일 제한은 "오늘 다 봤다! 내일 또 오자" 를 만들기 위함
- 10개 구문 저장은 "더 저장하고 싶은데..." 를 자연스럽게 유도
- 복습 게임 1종으로 재미를 맛보게 하되, 더 다양한 게임은 Premium
- 광고는 수익원이자 Premium 전환 동기

핵심은 **무료가 불쾌하면 안 된다**는 것. 무료 유저도 "이 앱 좋다"고 느껴야 한다.
제한은 있되, 제한이 학습을 방해하는 것이 아니라 "더 하고 싶은 욕구"를 만드는 방식이어야 한다.

## 4. 할인 규칙

### 4.1 상시 할인

| 할인 종류 | 대상 | 할인율 | 조건 |
|---|---|---|---|
| **연간 플랜** | 전체 | 41% (월간 대비) | 상시 적용 |
| **3개월 플랜** | 전체 | 20% (월간 대비) | 상시 적용 |
| **학생 할인** | 대학생/고등학생 | 연간 30% 추가 → 48,900원/년 | 학생증/이메일 인증 |

### 4.2 시점별 할인

| 시점 | 할인 내용 | 근거 |
|---|---|---|
| **첫 구독** | 첫 달 50% → 4,950원 | 진입 장벽 최소화, 체험 유도 |
| **7일 무료체험** | 연간 플랜에만 제공 | 업계 표준, 연간 전환 유도 |
| **월간→연간 업그레이드** | 사용 2~3개월 후 개인화 제안, 추가 10% 할인 | 이미 습관 형성된 유저 타겟 |
| **이탈 방지** | 구독 취소 시도 시 1개월 50% 할인 제안 | 이탈률 감소 |
| **복귀 유저** | 구독 만료 후 30일 내 → 연간 20% 추가 할인 | 윈백 캠페인 |

### 4.3 소셜/바이럴 할인

| 할인 종류 | 내용 | 효과 |
|---|---|---|
| **친구 초대** | 초대자 + 피초대자 각각 1개월 무료 | 바이럴 확산, CAC 절감 |
| **그룹 구독 (향후)** | 2인 연간 → 1인당 59,900원 (15% 할인) | 함께 학습 동기부여 |
| **리뷰 보상** | 앱스토어 리뷰 작성 시 Premium 3일 무료 | 평점 관리 |

### 4.4 할인 원칙 (반드시 지킬 것)

1. **할인은 "보상"이어야 한다** -- 유저가 뭔가를 했을 때 (친구 초대, 장기 구독, 학생 인증) 주는 것
2. **시간 압박 FOMO는 최소화** -- "24시간 내 결제하세요!" 같은 압박은 지양
3. **투명한 가격** -- 원래 가격과 할인 가격을 명확히 표시
4. **할인 후에도 수익성 확보** -- 최저 마진 라인 (연간 48,900원) 이하로는 불가

## 7. 의사결정 근거 (Rationale)

### 7.1 왜 월 9,900원인가?

- 케이크 Cake Plus 정가 16,000원보다 **38% 저렴** (유독 제휴가와 동일 수준)
- 말해보카 월간 19,500원의 **절반 수준**
- "만원 이하" 라는 심리적 저항선을 넘지 않음
- 한국 소비자에게 "9,900원" 은 "만원도 안 한다" 로 인식됨
- 스타벅스 아메리카노 2잔 가격 → "커피 2잔 참으면 영어 공부"

### 7.2 왜 연간 69,900원인가?

- 듀오링고 연간 ~83,000원보다 **16% 저렴**
- 말해보카 연간 119,000원보다 **41% 저렴**
- 스픽 연간 129,000원보다 **46% 저렴**
- 월 환산 5,825원 = "하루 194원" → 극도로 가성비 좋다는 인식
- "7만원 미만" 이라는 가격대는 한국 소비자에게 "부담 없는 자기투자"
- 월간 대비 41% 할인으로 연간 전환 동기 충분 (업계 기준 17% 할인이면 25~40% 전환, 41%면 더 높은 전환 기대)

### 7.3 왜 평생 199,000원인가?

- 연간 69,900원의 약 2.85배 → 3년 차부터 이득
- 야나두 평생 125만원 대비 극도로 저렴하다는 느낌
- 스픽 Premium Plus 연간 299,000원보다 저렴한 "평생" → 가치 인식 극대화
- 초기 현금흐름 확보에 도움
- 단, 무분별하게 팔면 장기 수익 감소 → 한정 판매 필수

### 7.4 왜 학생 할인 30%인가?

- 피클플러스 통한 말해보카 학생 할인도 30%
- 학생은 가격 민감하지만, 습관 형성 시 졸업 후 정가 전환 가능
- 연간 48,900원 → 월 4,075원 → "하루 136원" → 거부감 제로
- 학생 시절 형성된 브랜드 충성도는 장기 LTV로 환원

### 7.5 왜 3개월 플랜이 있는가?

- 연간은 너무 길고, 월간은 비싸 보이는 유저를 위한 징검다리
- "3개월 써보고 연간으로 갈아탈래" 라는 자연스러운 업그레이드 경로
- LingoDeer 등 성공 사례에서 3개월 플랜이 전환율 개선에 기여
- 학기 단위(3~4개월)와 맞아 학생 유저에게 특히 효과적

### 7.6 광고 수익 전략

- 무료 유저는 광고 시청 (배너 + 동영상 간 인터스티셜)
- 광고는 학습 흐름을 심하게 방해하지 않는 선에서 배치
- 보상형 광고: "광고 보고 하트 1개 충전" → 유저 선택권 부여
- Premium 전환 시 광고 완전 제거 → 강력한 업그레이드 동기

## 8. 리스크 및 대응 방안

| 리스크 | 대응 |
|---|---|
| 너무 싸서 수익 부족 | 가치 인식 높이기 (기능 강화) 후 점진적 가격 인상 |
| 무료가 너무 좋아서 유료 전환 안 됨 | Free 제한 조정 (A/B 테스트로 최적점 탐색) |
