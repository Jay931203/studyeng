## 2. 레벨별 콘텐츠 전략

### 2.1 B2 (Upper Intermediate) — 목표: +253개

B2 콘텐츠의 핵심 특성:
- 복잡한 문장 구조, 관용구 활용
- 전문 분야 기초 어휘 (법, 의학, 비즈니스, 정치)
- 자연스러운 연결 발화, 빠른 대화 템포
- 영국 영어/미국 영어 다양한 악센트

**적합한 콘텐츠 유형**:
1. 법률/정치 드라마 (복잡한 논쟁, 설득 화법)
2. 비즈니스/창업 콘텐츠 (전문 어휘 + 프레젠테이션)
3. TED Talks 전문 주제 (과학, 심리학, 경제)
4. 뉴스 인터뷰 및 패널 토론
5. SF/스릴러 영화 (복잡한 플롯 설명)
6. 영국 드라마/영화 (격식 표현 + 다양한 악센트)
7. 스탠드업 코미디 (말장난, 문화 레퍼런스)

### 2.2 C1 (Advanced) — 목표: +166개

C1 콘텐츠의 핵심 특성:
- 학술적/전문적 담론
- 미묘한 뉘앙스, 아이러니, 간접 표현
- 빠른 속도 + 줄임말 + 속어
- 문화적 맥락 이해 필요

**적합한 콘텐츠 유형**:
1. 학술 강의 및 대학 레벨 교육
2. 정치 풍자/시사 코미디 (John Oliver 스타일)
3. 다큐멘터리 내레이션 (BBC, PBS)
4. 토론 프로그램 및 정치 인터뷰
5. 문학 작품 영화화 (격식 대사 + 복잡한 감정 묘사)
6. 비즈니스 프레젠테이션 및 협상 장면
7. 전문 팟캐스트/인터뷰 (심층 대화)

### 2.3 C2 (Mastery) — 목표: +93개

C2 콘텐츠의 핵심 특성:
- 원어민 수준의 관용구, 속어, 은어
- 극도로 빠른 자연스러운 발화
- 고도의 문화적 레퍼런스
- 시/문학적 표현, 이중 의미, 풍자

**적합한 콘텐츠 유형**:
1. 즉흥 코미디 (완전 무대본, 초고속)
2. 래퍼/힙합 아티스트 인터뷰 (속어 집중)
3. 영국 패널쇼 (빠른 위트, 문화 유머)
4. 학술 토론 및 철학 강의
5. 시/문학 낭독
6. 법정 실제 변론 (법률 영어 최고난도)
7. 고급 풍자 프로그램

## 7. 소싱 가이드라인

### 7.1 YouTube 영상 선별 기준

1. **음질**: 배경음악이 대사를 압도하지 않는 클립 선별
2. **길이**: 30초-3분 클립 (Whisper 전사 최적 구간)
3. **대사 밀도**: 무음 구간이 전체의 30% 미만
4. **화자 수**: 1-3인 대화 선호 (4인 이상은 화자 구분 어려움)
5. **자막 불필요**: Groq Whisper로 자체 전사하므로 YouTube 자막 무관
6. **저작권**: YouTube embed (IFrame API) 사용이므로 모든 공개 영상 사용 가능

### 7.2 CEFR 태깅 검증

자동 태깅 후 수동 검증 필수:
- B2: 관용구 3개 이상 또는 전문 어휘 5개 이상 포함
- C1: 복합 문장 구조 50% 이상 또는 학술 어휘 5개 이상
- C2: 속어/은어 3개 이상 또는 문화 레퍼런스 집중

### 7.3 Expression Dictionary 확장

신규 B2-C2 영상 추가 시 expression-entries-v2.json도 동시 확장 필요:
- 목표: B2 표현 +200개, C1 +150개, C2 +80개
- 영상 당 평균 2-3개 신규 표현 추출

## 11. 6-Level CEFR 시스템과의 연계

이 콘텐츠 확보 플랜은 `six-level-tier-integration.plan.md`의 6단계 CEFR 시스템 구현과 병행되어야 한다.

- **B2 유저** (중상급): B1-B2 콘텐츠 추천 → B2 콘텐츠 400개 필요
- **C1 유저** (고급): B2-C1 콘텐츠 추천 → C1 콘텐츠 200개 필요
- **C2 유저** (마스터): C1-C2 콘텐츠 추천 → C2 콘텐츠 100개 필요

6레벨 시스템에서 CEFR +/-1 범위 추천 필터가 적용되므로, 각 레벨에 최소 100개 이상의 콘텐츠가 있어야 유저가 "볼 게 없다"고 느끼지 않는다.
