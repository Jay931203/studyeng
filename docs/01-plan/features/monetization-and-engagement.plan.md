## 1. Overview

### 1.1 Purpose

현재 MVP로 구현된 영어 학습 앱(TikTok 스타일 비디오 피드 + 자막 + 미니게임 + 게이미피케이션)을
상업적으로 지속 가능한 제품으로 전환하기 위한 수익화 전략 및 인게이지먼트 시스템을 정의한다.

목표는 단순한 "교육 앱"이 아닌, 유저가 자발적으로 매일 열고 싶어지는 콘텐츠 소비 플랫폼이다.

### 1.2 Background

- 한국 영어 학습 앱 시장은 연간 수천억 원 규모 (Duolingo 한국 MAU 수백만)
- 기존 경쟁 앱들은 학습 압박, 복잡한 커리큘럼, 지루한 UX로 이탈률이 높음
- StudyEng의 차별점: "공부하는 느낌 없이 영어에 노출" - 이 철학이 수익화 전략의 기반

## 4. Freemium Model - 무료 vs 유료 기능 경계

### 4.1 핵심 원칙

무료 티어는 "맛보기"가 아니라 진짜 가치를 제공해야 한다.
유저가 "이건 돈 내도 아깝지 않겠다"는 확신이 생겼을 때 자연스럽게 전환하도록 설계.

### 4.2 기능별 티어 정의

| 기능 | 무료 (Free) | 프리미엄 (9,900원/월) |
|------|-------------|----------------------|
| 일일 영상 시청 | 10개/일 | 무제한 |
| 자막 모드 | 한국어 자막 없음 (영어 only) | 영어 + 한국어 자막 |
| 단어 의미 팝업 | 없음 | 영상 중 단어 터치 -> 뜻 팝업 |
| 미니게임 종류 | Fill-in-blank, Sentence Puzzle (2종) | 전체 4종 + AI 적응형 난이도 |
| A-B 구간 반복 | 3회/영상 제한 | 무제한 |
| 구문 저장 | 20개 한도 | 무제한 |
| 저장 구문 복습 | 불가 (저장만) | 저장 구문으로 게임 생성 |
| 광고 | 영상 5개마다 배너 | 광고 없음 |
| 다운로드/오프라인 | 불가 | 영상 스크립트 오프라인 저장 |
| 시리즈 접근 | 기본 시리즈 (카페/공항 영어 등) | 드라마/영화 시리즈 전체 |

**연간 구독**: 79,900원/년 (월 6,658원 - 33% 할인)

### 4.3 전환 유도 트리거 (Upgrade Prompt 시점)

전환 압박은 금물. 자연스러운 "아 이게 있으면 좋겠다" 모멘트에만 표시:
- 한국어 자막 버튼 터치 시 (가장 효과적)
- 저장 구문 20개 초과 시
- Friends 에피소드 접근 시도 시
- A-B 반복 3회 소진 시

## 5. Engagement Hooks - 일일 재방문 유도

### 5.1 핵심 원칙

Duolingo는 "공부 안 하면 streak 끊긴다"는 두려움을 사용 - StudyEng은 이 방식을 거부.
대신 "오늘 뭔가 재미있는 게 기다리고 있다"는 기대감으로 접근.

### 5.2 Daily Engagement 메커니즘

**1. 오늘의 영상 (Daily Featured)**
- 매일 오전 7시 새 영상 1개 "Today's Pick" 배너
- 실제 영어권 트렌드, 밈, 인기 표현 활용
- 48시간 후 사라짐 -> 희소성 + FOMO

**2. 연속 접속 보상 (Streak 재정의)**
- 기존 streak: "어제 안 했으면 리셋" -> 압박
- StudyEng streak: "이번 주 N일 접속" 주간 기준
- 7일 완주 -> 특별 배지 + 프리미엄 3일 무료 체험
- 하루 빠져도 streak 유지 (1회 면제권 자동 제공)

**3. 미완성 영상 알림**
- "어제 반쯤 보다 멈춘 영상이 있어요" 알림
- 영상 중간에 그만두면 자동 북마크
- 학습 압박 없이 "잠깐, 5분만"으로 유입

**4. 소셜 훅 (친구 없이도 작동)**
- 이번 주 가장 많이 저장된 표현 TOP 3 공개
- "이 표현 아는 사람? 78%가 틀렸어요" 공유 카드
- 카카오톡 공유 -> 초대 리워드 (둘 다 XP 보너스)

**5. 컬렉션 완성 욕구**
- 시리즈별 진행도 표시: "Friends 시즌1 - 4/20 에피소드"
- 카테고리별 뱃지: "카페 영어 마스터" "공항 영어 완료"
- 완성 욕구를 학습 동기로 전환

**6. 페르소나 기반 개인화**
- 온보딩 시 "어떤 영어가 필요해요?" 3지선다
  - 여행/일상 대화 / 직장/비즈니스 / 미드/영화 즐기기
- 피드 순서를 선택 기반으로 조정 (AI 비가시적 작동)
- "당신이 좋아할 영상" 섹션 -> 클릭률 증가

### 5.3 알림 전략

| 알림 유형 | 시점 | 메시지 예시 |
|-----------|------|------------|
| 일일 영상 알림 | 오전 9시 (설정 가능) | "오늘의 Friends 표현 - '내가 낼게' 영어로?" |
| 미완성 영상 | 다음날 오후 | "어제 보다 멈춘 영상이 기다려요 (1분 남음)" |
| 주간 streak | 일요일 오후 | "이번 주 5일 달성! 배지 받으러 가기" |
| 트렌드 표현 | 주 1-2회 | "요즘 미국에서 유행하는 표현, 알고 계세요?" |

규칙: 알림은 일 1회 한도. 유저가 끈 알림은 절대 재활성화 요청 없음.

## 6. Content Strategy - 18개 시드 영상 이후 확장

### 6.1 콘텐츠 피라미드

```
[Level 3] 오리지널 콘텐츠 (Phase 3)
  - 자체 제작 숏폼 영상
  - 크리에이터 파트너십

[Level 2] 파트너십 콘텐츠 (Phase 2)
  - 한국 유튜버 영어 학습 채널 협력
  - 영어권 인기 Shorts 큐레이션 허가

[Level 1] YouTube 임베드 큐레이션 (MVP 현재)
  - 시드 18개 + 주 3-5개 추가
  - 카테고리별 플레이리스트 확장
```

### 6.2 단기 콘텐츠 확장 (0-3개월: YouTube 임베드)

**우선 확장 카테고리 (검증된 관심 기반)**

| 카테고리 | 목표 영상 수 | 소스 |
|----------|-------------|------|
| 일상 대화 (카페/식당/쇼핑) | 30개 | YouTube Shorts, Reels |
| 드라마 명장면 (Friends, 오피스) | 40개 | 클립 업로더 채널 |
| 트렌드/밈 표현 | 20개 | 주간 업데이트 |
| 비즈니스 영어 | 20개 | TED-Ed, 비즈니스 채널 |
| 여행 영어 (공항/호텔/관광) | 20개 | 여행 브이로그 |

**콘텐츠 큐레이션 파이프라인 (내부 도구)**
1. 영상 URL 입력
2. YouTube Caption API로 자막 자동 추출
3. Claude API로 난이도 분류 + 태그 생성
4. 관리자 검수 후 게시
목표: 편당 큐레이션 시간 15분 이하

### 6.3 중기 콘텐츠 전략 (3-6개월)

**UGC(User Generated Content) 요소 추가**
- 유저가 YouTube URL 제안 -> 관리자 승인 후 게시
- "이 영상으로 공부하고 싶어요" 버튼
- 제안 채택 시 해당 유저에게 프리미엄 1주일 보상

**시즌제 콘텐츠**
- 분기별 테마 시리즈: "할로윈 영어", "크리스마스 표현", "2025 슬랭"
- 시즌 한정 뱃지 -> 수집 욕구

### 6.4 자막 데이터 품질 전략

YouTube 자동 자막은 오류율 높음:
1. YouTube Caption API 추출 (기본)
2. Claude API로 오류 교정 + 한국어 번역 생성
3. 유저 피드백 버튼 ("자막 오류 신고") -> 수정 큐
4. 신고 누적 시 자동 재처리

## 8. Key Metrics to Track - 성공 지표

### 8.1 북극성 지표 (North Star Metric)

**"주 3일 이상 활성 유저 수 (Weekly Active Learners)"**

이유: DAU는 알림 열기로 부풀릴 수 있음. 실제 영상을 본 유저가 핵심.

### 8.2 Acquisition (유입)

| 지표 | 목표 | 측정 도구 |
|------|------|----------|
| 신규 가입 (월) | 출시 1개월 500명, 6개월 2,000명 | Supabase Auth |
| 채널별 유입 비율 | 카카오공유 > 앱스토어 > 검색 | UTM 파라미터 |
| 온보딩 완료율 | 80% 이상 | Funnel 분석 |

### 8.3 Activation (활성화)

| 지표 | 목표 | 측정 도구 |
|------|------|----------|
| 첫 영상 시청 완료율 | 70% 이상 | 영상 완료 이벤트 |
| 첫 미니게임 완료율 | 50% 이상 | GameResult 테이블 |
| 첫 구문 저장 | 40% 이상 | SavedPhrase 테이블 |

### 8.4 Retention (재방문)

| 지표 | 목표 | 측정 방법 |
|------|------|----------|
| D1 리텐션 | 50% 이상 | 다음날 재방문 유저 |
| D7 리텐션 | 30% 이상 | 7일 후 재방문 유저 |
| D30 리텐션 | 15% 이상 | 30일 후 재방문 유저 |
| 주간 streak 7일 달성률 | 20% 이상 | UserProgress 테이블 |

### 8.5 Revenue (수익)

| 지표 | 목표 | 측정 도구 |
|------|------|----------|
| 무료 -> 유료 전환율 | 3-5% | 결제 DB |
| 월간 구독 유지율 | 70% 이상 | Churn 추적 |
| LTV (고객 생애 가치) | 60,000원 이상 | 평균 구독 기간 x 월 요금 |
| ARPU (유저당 평균 수익) | 500원/월 | 전체 수익 / MAU |

### 8.6 Engagement Quality (핵심 - 공부하는 느낌 없음 지표)

| 지표 | 목표 | 의미 |
|------|------|------|
| 평균 세션 영상 시청 수 | 3개 이상 | 자발적 계속 시청 |
| 게임 자발 참여율 | 60% 이상 | 강제 없이 게임 선택 |
| 알림 비활성화율 | 30% 이하 | 알림이 유용하다고 느낌 |
| 앱스토어 리뷰 평점 | 4.3 이상 | "재미있다" 언급 비율 |

## 9. Success Criteria

### 9.1 Definition of Done (이 플랜 완료 기준)

- [ ] 프리미엄 티어 기능 게이팅 구현 완료
- [ ] Stripe 또는 카카오페이 구독 결제 연동
- [ ] 일일 재방문 알림 시스템 작동
- [ ] 콘텐츠 큐레이션 내부 도구 완성 (URL -> 게시 파이프라인)
- [ ] 핵심 지표 대시보드 구축 (Supabase Analytics)

### 9.2 Quality Criteria

- [ ] 무료 유저도 매일 의미 있는 경험 가능 (10개 영상 + 2종 게임으로 충분)
- [ ] 업그레이드 프롬프트가 방해가 되지 않음 (세션당 최대 1회)
- [ ] 구독 취소가 쉬움 (앱 내에서 1단계로 취소 가능)

## 10. Risks and Mitigation

| Risk | Impact | Likelihood | Mitigation |
