# Shortee (슏티) 경쟁 환경 분석 보고서

**작성일:** 2026년 3월 9일
**대상:** 한국 영어학습 앱 시장
**목적:** Shortee의 시장 진입 전략 수립을 위한 경쟁사 분석

### 2.1 Cake (케이크) -- 가장 직접적인 경쟁자

**핵심 정보**
- **개발사:** 케이크(주) (네이버 > 스노우 > 케이크, 2020년 분사)
- **출시:** 2018년 3월
- **누적 다운로드:** 1억 7천만건
- **글로벌 MAU:** ~1,000만 (언어학습 앱 세계 2위)
- **해외 이용자 비중:** 90% 이상
- **지원 언어:** 20개

**가격 모델**
- 무료: 하트 제한, 광고 포함, 제한된 콘텐츠
- 케이크 플러스: 월 $13.99 (약 18,000원) / LG U+ 유독 월 9,900원
- 7일 무료 체험 제공
- 네이버 플러스 멤버십 스튜던트 연동 (대학생 할인)

**핵심 기능**
- 드라마/영화/브이로그 등 실생활 영상 클립 기반 학습
- 매일 새로운 영어 표현 업데이트 (무료)
- AI 발음 피드백
- 퀴즈 기반 반복 학습
- 2022년 하이브 에듀(HYBE EDU) 인수 -> K-pop 아이돌 IP 활용

**강점 (Shortee가 배울 점)**
- 영상 클립 기반 학습의 원조 -- "짧은 영상으로 영어 배우기" 컨셉을 시장에 각인시킴
- 매일 무료 콘텐츠 업데이트로 복귀 유도
- K-pop IP를 활용한 글로벌 확장 (BTS, 블랙핑크 등)
- 높은 앱스토어 평점 (4.7~4.9점)
- 간편한 인터페이스, 진입장벽 낮음

**약점 (Shortee가 공략할 기회)**
- **한국 시장 입지 약화:** 2024년 한국 MAU 상위 5위에서 탈락
- **학습 깊이 부족:** 사용자들이 케이크 하나만으로는 충분하지 않다고 느낌 -> 다른 앱 병용
- **비싼 구독료:** 월 $13.99는 스픽(월환산 ~10,750원), 듀오링고(월환산 ~7,250원) 대비 높음
- **하트 제한 시스템:** 무료 사용자 경험 저하
- **피상적 반복:** 표현 암기에 머물고 깊이 있는 학습으로 연결이 안 됨
- **콘텐츠 개인화 부족:** 모든 사용자에게 같은 콘텐츠 -> "나에게 안 맞음" 이탈

**사용자 불만 요약**
1. 유료 구독 없이는 제대로 학습 불가 (하트, 광고, 저장 제한)
2. 혼자서는 학습이 완결되지 않음 -> 다른 앱과 병용 필요
3. 반복학습 구조의 단순함
4. 월 $13.99의 가격 부담

### 2.2 Duolingo (듀오링고) -- 게이미피케이션 벤치마크

**핵심 정보**
- **본사:** 미국 피츠버그
- **글로벌 MAU:** ~1.35억 (2025 Q3)
- **한국 MAU:** 62만 (2024.11, 전년 대비 244% 성장)
- **한국 누적 다운로드:** ~500만건
- **한국 누적 매출:** ~1,000만 달러
- **한국 월 최고 매출:** 67만 달러 (2024.10)
- **글로벌 연매출:** ~10억 달러 (2025 전망)
- **한국어 기반 영어 학습자:** 297만명 (2025.12)

**가격 모델**
- 무료: 하트(에너지) 제한, 광고 포함
- Super (슈퍼): 연 87,000원 (월환산 ~7,250원) -- 무제한 하트, 광고 제거
- Max (맥스): 더 높은 가격 -- AI 회화, 영상 코칭 등 프리미엄 AI 기능

**핵심 기능**
- 게이미피케이션의 교과서: XP, 스트릭, 리그, 리더보드
- 하트/에너지 시스템 (2025.7 개편)
- GPT-4 기반 AI 튜터 (Max 한정)
- 듀오링고 스코어 (학습 진전도 지표)
- 172개 언어 과정 (2025년 역대 최대 확장)

**강점 (Shortee가 배울 점)**
- **게이미피케이션 끝판왕:** XP, 스트릭, 리그 등이 학습 지속 습관을 만듦
- **무료 모델의 강력함:** 기본적으로 무료 -> 진입장벽 제로 -> 바이럴
- **매일 돌아오게 만드는 구조:** 스트릭 유지 동기, 리그 경쟁심리
- **캐릭터 브랜딩:** 듀오(부엉이)의 밈화 -> 앱 밖에서도 바이럴
- **한국 시장 급성장:** 1년 만에 MAU 244% 성장은 경이적

**약점 (Shortee가 공략할 기회)**
- **실제 영어 콘텐츠 부재:** 드라마/영화가 아닌 인위적 문장으로 학습 -> "실생활 영어와 거리감"
- **학습 콘텐츠의 깊이 부족:** 문장 번역, 듣기, 따라하기가 반복 -> 고급 학습자에게 지루함
- **하트/에너지 시스템 불만:** 틀리면 학습이 멈춤 -> 좌절감 유발
- **광고 과다:** 무료 사용자에게 광고가 학습 흐름을 끊음
- **가격 정책 혼란:** Super vs Max 구분, 국가별 가격 차이, 업그레이드 압박
- **한국어 과정의 한계:** 영어 기반에 비해 콘텐츠 품질이 떨어짐
- **공부하는 "느낌":** 게이미피케이션이 있지만 본질적으로 "문제 풀기"에 가까움

**사용자 불만 요약**
1. 하트가 떨어지면 학습 중단 -> 6시간 대기 or 과금
2. "무료"지만 실질적으로 유료를 유도하는 구조
3. Max로의 업그레이드 압박
4. 실생활 영어와 괴리 -- "듀오링고만으로 영어를 할 수 있게 되진 않는다"
5. 음성 품질의 편차

### 2.3 Speak (스픽) -- AI 회화 강자

**핵심 정보**
- **개발사:** Speakeasy Labs (미국 실리콘밸리)
- **한국 법인:** 스픽이지랩스코리아(주)
- **출시:** 2019년 12월
- **한국 누적 다운로드:** 500만+ (2024.1)
- **한국 MAU:** 24만 (2024.11)
- **한국 연 매출:** ~960만 달러 (센서타워 추정)
- **한국 매출 비중:** 90.9% (한국이 핵심 시장)
- **기업가치:** $10억 (유니콘, 2024.12)
- **투자:** OpenAI 스타트업 펀드 시리즈 B 380억원 (2022)

**가격 모델**
- 무료: 제한된 레슨
- 프리미엄: 월 29,000원 / 연 129,000원 (스픽 튜터 제한적)
- 프리미엄 플러스: 연 299,000원 (프리톡, 스픽 튜터 무제한)
- 평생: 450,000원

**핵심 기능**
- AI 기반 실시간 발음 분석 (정확도, 억양, 유창성)
- Learn -> Practice -> Apply 3단계 학습 구조
- 스픽 튜터: OpenAI GPT 기반 AI 개인교사
- 프리톡: AI와 주제별 자유 대화
- 500일 분량 커리큘럼, 2,000개+ 학습 콘텐츠
- 상황별 회화 시나리오 (여행, 비즈니스, 면접 등)

**강점 (Shortee가 배울 점)**
- **"말하기"에 올인한 포지셔닝:** 명확한 USP
- **OpenAI 파트너십:** 기술적 해자
- **한국 시장 집중:** 한국 매출 90%+ -> 한국 사용자 이해도 높음
- **가성비 인식:** 월 1~2만원대로 "원어민 과외" 대체
- **커리큘럼 체계화:** 왕초보~고급까지 500일 분량 -> 장기 학습 동기

**약점 (Shortee가 공략할 기회)**
- **공공장소 사용 불가:** 음성 입력 필수 -> 지하철/카페에서 사용 못 함
- **내성적인 사용자 배제:** 말하기가 부담스러운 한국인이 많음
- **문법/리딩 학습 부족:** 스피킹에만 특화
- **비싼 프리미엄 플러스:** 연 299,000원은 부담
- **결제 복잡성:** 인앱결제 vs 홈페이지 결제 혼란, 해외결제 수수료 문제
- **커리큘럼 변경 불만:** 업데이트로 난이도가 갑자기 바뀌어 기존 사용자 불만
- **발음 교정의 한계:** 발음을 "교정"하기보다 단순 "평가"에 그침

**사용자 불만 요약**
1. 대중교통에서 사용 불가 -- 가장 큰 허들
2. 문법/독해 학습 불가 -> 다른 앱 병용 필요
3. 연간 결제만 가성비 있음, 월결제는 비쌈
4. 업데이트 때마다 커리큘럼이 바뀌어 혼란
5. 발음 교정이 피상적

### 2.4 RedKiwi (레드키위) -- 가장 유사한 직접 경쟁자

**핵심 정보**
- **개발사:** 하얀마인드 (대전)
- **출시:** 2017년 (미미킹 -> 레드키위 리뉴얼)
- **글로벌 다운로드:** 30만+
- **주요 시장:** 일본 > 한국 (일본 다운로드가 더 많음)
- **규모:** 스타트업 수준 (케이크, 스픽 대비 매우 작음)

**가격 모델**
- 무료: 1시간당 10분 학습 (쿨타임 제한)
- 베이직/프리미엄 구독: 1개월/3개월/12개월
- 3일 무료 체험
- 친구 추천 시 7일 프리미엄 제공

**핵심 기능**
- 유튜브 영상 기반 학습 (영화, TV쇼, 뉴스, 셀럽 인터뷰 등)
- 듣기 퀴즈 (문장을 듣고 단어 배열) -- 가장 핵심 기능
- 쉐도잉 (녹음 & 비교)
- 워드맵 (단어 수집 & 관리)
- 100% 사람 번역 (기계 번역 사용 안 함)
- Q&A (유료 사용자 질문, 레드키위 측 답변)

**강점 (Shortee가 배울 점)**
- **Shortee와 가장 유사한 컨셉:** 유튜브 영상 기반 학습
- **사람 번역의 품질:** 기계 번역 대비 자연스러움 -> Shortee도 Claude 번역으로 차별화 가능
- **듣기 퀴즈의 중독성:** 단어 배열 퍼즐이 게임적 요소
- **워드맵:** 단어 수집이 시각적으로 보이는 재미
- **장기 사용자 존재:** "3년째 사용 중" 같은 후기 -> 충성도 높은 코어 사용자 존재

**약점 (Shortee가 공략할 기회)**
- **극소규모:** 30만 다운로드 -> 시장 임팩트 미미
- **시간 제한 모델:** 1시간당 10분은 매우 짧음 -> 학습 몰입 방해
- **UX 올드함:** 2017년 출시 이후 UI/UX 현대화 부족
- **기능 단조로움:** 듣기 퀴즈 + 쉐도잉 이외의 활동이 제한적
- **마케팅 부재:** 인지도가 매우 낮음
- **일본 편중:** 한국보다 일본에서 더 인기 -> 한국 시장 투자 부족

**Shortee와의 직접 비교:**

| 항목 | RedKiwi | Shortee (계획) |
|------|---------|---------------|
| 콘텐츠 소스 | YouTube 영상 | YouTube 쇼츠/클립 |
| UX 철학 | 학습 도구 | 놀이 (Play First) |
| 학습 활동 | 듣기 퀴즈, 쉐도잉 | 터치 기반 게임들 |
| 번역 | 100% 사람 번역 | Whisper + Claude AI |
| 음성인식 | 쉐도잉 (있음) | 없음 (의도적 제거) |
| 피드 방식 | 카테고리 탐색 | 숏폼 피드 (TikTok식) |
| 무료 모델 | 시간 제한 (10분/시간) | 프리미엄 + 광고 하이브리드 |

### 2.5 기타 경쟁자

#### YouGlish (유글리시)
- **성격:** 발음 검색 도구 (학습 앱이라기보다 참고 도구)
- **콘텐츠:** YouTube 자막 기반 1억개+ 영어 클립
- **한국 사용자:** 미미 (정확한 수치 미확인)
- **가격:** 무료 + 프리미엄 (광고 제거 등)
- **Shortee와의 관계:** 직접 경쟁이 아닌 보완적 도구. Shortee가 "실시간 발음 확인"을 넣으면 이 영역까지 커버 가능

#### EWA
- **성격:** 영화/TV/책 기반 영어 학습
- **핵심:** 영화 클립 + 10,000권 영어 책 + 미니게임
- **가격:** 월 ~$10
- **한국 인지도:** 매우 낮음
- **강점:** 콘텐츠(영화, 책) 다양성이 높고, UI가 세련됨
- **약점:** 결제 관련 불만 심각 (자동 과금, 취소 어려움, Trustpilot 불만 다수)
- **시사점:** 콘텐츠 기반 영어 학습의 가능성을 보여주지만, 결제 UX가 앱을 죽일 수 있음

#### ELSA Speak (엘사스픽)
- **성격:** AI 발음 교정 특화
- **핵심:** 음성인식 기반 발음 정확도/억양/유창성 분석
- **가격:** 월 $13.33 (Premium)
- **타겟:** 중급~고급 학습자, 토익스피킹/IELTS 준비
- **강점:** 발음 교정에서 업계 최고 수준
- **약점:** 발음"만" 가능 -> 종합 영어학습 불가, 음성인식 필수
- **Shortee와의 관계:** 직접 경쟁 아님. Shortee는 "음성인식 안 함"이 원칙

#### 말해보카
- **성격:** 영단어 암기 앱 (스피킹 결합)
- **한국 MAU:** 29만 (2024.11, 2위)
- **한국 매출:** 교육 앱 1위 (RPD $8, 업계 최고)
- **강점:** 한국 사용자에게 최적화, 높은 수익성

## 6. 전략적 권장사항

### 6.1 Shortee의 포지셔닝 전략

**핵심 메시지:** "영어는 게임이다 -- 숏폼 영상으로 매일 30초씩 놀면 영어가 늘어난다"

**타겟 세그먼트 (우선순위):**
1. **1순위:** 스픽을 써봤지만 "말하기가 부담스러워서" 이탈한 사용자
2. **2순위:** 듀오링고를 쓰지만 "실제 영어와 다르다"고 느끼는 사용자
3. **3순위:** 케이크를 썼지만 "깊이가 없다"고 느끼고 떠난 사용자
4. **4순위:** 영어 앱 자체를 처음 쓰는 MZ세대

### 6.2 경쟁사별 대응 전략

**vs 케이크:**
- "케이크처럼 영상으로 배우지만, 게임으로 더 재미있게"
- 케이크의 하트 제한/구독 유도를 공격 -> 더 관대한 무료 모델
- 케이크에 없는 "게임적 학습 활동" 강조

**vs 듀오링고:**
- "듀오링고의 게이미피케이션 + 실제 영어 콘텐츠"
- 인위적 문장 대신 진짜 드라마/영화 장면
- 스트릭/경쟁 요소는 벤치마킹하되, 하트/에너지 제한은 안 함

**vs 스픽:**
- "말하기 없이도 영어가 느는 앱"
- 지하철/카페/어디서든 사용 가능함을 강조
- 내성적인 한국인 사용자 + 공공장소 사용자 공략

**vs 레드키위:**
- 같은 YouTube 기반이지만, UX를 "숏폼 피드 + 게임"으로 완전히 차별화
- 레드키위의 시간 제한(10분/시간) 대비 더 관대한 모델
- 최신 UI/UX vs 레드키위의 올드한 인터페이스

### 6.3 듀오링고에서 배워야 할 것들

1. **스트릭 (연속 학습일):** 매일 돌아오게 만드는 가장 강력한 장치
2. **리그/리더보드:** 경쟁심리 자극 -> 학습 지속
3. **캐릭터 브랜딩:** Shortee만의 마스코트/캐릭터 -> 밈 가능한 캐릭터
4. **무료 우선 모델:** 충분히 즐길 수 있는 무료 -> 자연스러운 유료 전환
5. **알림의 기술:** 듀오링고의 푸시 알림은 업계 최고 수준 (유머러스하고 비강제적)

### 6.4 피해야 할 것들 (경쟁사 실패에서 배움)

1. **하트/에너지 제한 시스템:** 사용자 좌절감 유발 (듀오링고, 케이크 최대 불만)
2. **시간 제한 모델:** 학습 몰입 파괴 (레드키위 최대 약점)
3. **과도한 구독 유도:** "무료"라고 광고하면서 실질적으로 유료 유도하면 신뢰 상실

6. **불투명한 결제:** 자동 과금, 취소 어려움 -> 앱 평점 폭락 (EWA 사례)

## 8. 결론

### Shortee의 시장 진입 근거

1. **시장이 빠르게 성장 중:** 한국 영어학습 앱 시장은 연 50%+ 성장하고 있으며, MAU 기준으로 2024년에 전년 대비 90% 증가. 파이가 커지고 있다.

2. **영상 기반 학습의 강자가 약해지고 있다:** 케이크가 한국 MAU 상위 5위에서 탈락. 영상 기반 영어학습의 수요는 있지만, 케이크가 그 수요를 충족시키지 못하고 있다.

3. **"말하기 안 해도 되는 앱"의 빈 자리:** 스픽, ELSA 등 음성인식 기반 앱이 강세이나, "말하기가 부담스러운" 사용자 세그먼트는 여전히 미개척. 한국인의 내성적 성향 + 대중교통 문화를 고려하면 이 세그먼트는 크다.

4. **숏폼 UX가 아직 없다:** 영어학습 앱 중 TikTok/릴스 같은 숏폼 피드 UX를 채택한 앱이 없다. Shortee가 최초.

5. **게이미피케이션은 검증되었다:** 듀오링고의 244% MAU 성장이 증명. 다만 듀오링고는 "실제 콘텐츠"가 없다. Shortee는 "실제 영상 + 게이미피케이션"의 조합.

### 최우선 과제

- **콘텐츠 양 확보:** 현재 701개 -> 최소 2,000개+ 영상 확보 필요
- **게임 다양화:** 현재 2종 -> 5종 이상의 터치 기반 미니게임 개발
- **습관 형성 장치:** 스트릭, 일일 미션, 리워드 시스템 구축
- **무료 경험 최적화:** 과금 전환 전에 충분한 "재미"를 무료로 제공
